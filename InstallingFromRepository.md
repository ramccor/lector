This project use Mercurial repository

## Getting the sources ##
```
hg clone https://code.google.com/p/lector/
```

## Using the XMLs  via Makefile ##
Now we have to move to lector/ and do some work:

#### linux ####
```
make all
```

#### windows ####
```
win_make.bat
```


## Start! ##
You're ready now:
```
./lector.pyw
```

or:
```
python lector.pyw
```
